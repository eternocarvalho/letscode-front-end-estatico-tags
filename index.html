Tags HTML

Tag <p>
Essa Tag cria um elemento de parágrafo.
Tag <img>
Permite inserir uma imagem no corpo do texto.
Tag <input>
Cria um campo a ser preenchido pelo usuário.
Tag <blockquote>
Usada pra citações longas.
Tag <q>
Usada para citações curtas.
Tag <adress>
Usada para referenciar contatos.
Tag <h1>
Criação de um título. Importante para uma boa semântica do código.
Tag <ol>
Usada para listas ordenadas.
Tag <ul>
Usada para listas não-ordenadas.
Tag <a>
Tag de ancoramento.
Tag <title>
Define o título da página.
